# Feynman

A simple HTTP proxy written in ruby capable of on-demand back-end summoning

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'feynman'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install feynman
